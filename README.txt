INTRODUCTION
------------

The Sticky Toolbar module allows each user to separately toggle a setting to 
make core's toolbar sticky.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/sticky_toolbar
 
 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/sticky_toolbar


REQUIREMENTS
------------

 * This module requires no modules outside of Drupal core.


INSTALLATION
------------
 
 * Install the Sticky Toolbar module as you would normally install a
   contributed Drupal module. Visit: 
   https://drupal.org/documentation/install/modules-themes/modules-8
   for further information.


CONFIGURATION
-------------
 
    1. Navigate to Administration > Extend and enable the module.

    2. Configure user permissions in Administration > People > Permissions:

       - Use Sticky Toolbar

       Toggle settings require this permission to be accessible. The toggle
       setting page will be empty unless this permission is granted.

    3. To toggle the settings, navigate to
    Administration > Configuration > User interface > Sticky Toolbar settings.
